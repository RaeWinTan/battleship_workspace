  1. Setting Parameters
    1. Grid Size: set the boards to a N x N size square
    2. No. of Ships: set the number of ships that will be on the boards
    3. No. of Iterations: number of games the bots will be playing
  2. Chart data points
    1. Click on the data point to see how the game was played by the two bots.
      The visualisation is not meant to support a large grid size(>25) though the game supports it. This visualisation is just to prove that the algorithm(smart bot) is working the way it suppose to.
    2. Data points on the chart represent the number of turns that cooresponding bot took
Smart bot took {{eff}}% less turns to shoot down all opponent's ships compared to dumb bot.
